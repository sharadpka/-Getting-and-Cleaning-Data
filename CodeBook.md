# Code Book
This code book describes the source of the data and the different variables involved in the file provided

## Source
The data was provided by the UCI Machine Learning Repository and is about Human Activity Recognition collected using Smartphones.
These are experiments carried out on 30 subjects/ participants carried out while performing 6 different activities.

## Activities
The protocol of activities performed by these participants composed of 6 basic routines as follows:
1. WALKING
2  WALKING_UPSTAIRS
3  WALKING_DOWNSTAIRS
4  SITTING
5  STANDING
6  LAYING

The above activity labels also follow the same acivity ID and/or activity label pair as in the dataset.
